## 1. Context

The Alphabet section is 4 games: **LГҰr Alfabetet** (calm browse, `/alphabet/learn`), **Bogstav Quiz**
(`/alphabet/quiz`), **Hukommelse 10 & 20** (letters memory, `/learning/memory/letters/10|20`). Mechanically they
are solid and the owner is happy with how they *teach*; the gap is purely **look & feel** vs the Structured-World
shell (per PRD-06 В§1). Alphabet is the simplest section to re-skin cleanly, so it proves the Foundation language.

**What's flat today (grounded this session):** Bogstav Quiz shows a **flat 2D system emoji** (рҹҗ’/рҹҡҖ/рҹҚҺ) inside the
frosted `PromptStage` card, with keyboard-key letter tiles. The browse bloom and memory cards likewise show system
emoji. All become the tactile language + baked soft-3D art.

**Content finding (must handle):** there are **two independent, disagreeing** letterвҶ’wordвҶ’emoji manifests:
- `src/config/letterWords.ts` вҶ’ `LETTER_WORDS` (25 letters; omits Q/W/X/Г…) вҖ” used by **Bogstav Quiz + LГҰr Alfabetet**.
- `src/components/learning/MemoryGame.tsx` вҶ’ inline `LETTER_ICONS` (all 29; different words/emoji) вҖ” used by **Memory**.

They diverge (A = Abe рҹҗ’ vs Ananas рҹҚҚ; B = Bil рҹҡ— vs BjГёrn рҹҗ»; D = Drage рҹҗү vs Due рҹ•ҠпёҸ; вҖҰ). See В§4. **Decision for the
owner (В§6):** reconcile to ONE shared manifest so a letter shows the *same* baked object everywhere (recommended вҖ”
consistency is a real quality win for a 5-year-old, and it halves the art), vs keep two sets and bake both.

**Success:** entering Bogstav Quiz feels like the alphabet menu it launched from вҖ” the word-picture is a tactile
soft-3D object resting in the calm rainbow/ocean/space/dino world, the letter tiles are pressable clay, the letter
glyphs crisp and central. The letter (the lesson) is as clear as today, or clearer.

### 3.2 Bogstav Quiz
- **No config/mechanics change** вҖ” `generateQuizItem`/`generateOptions`/`round`/`gameId`/hint/audio all stay.

surgery. No mechanic/flow changes proposed. (If play-testing the clay tiles reveals the lip-less tile reads as less
pressable, apply the Foundation's subtle-edge fallback вҖ” F1 note.)
